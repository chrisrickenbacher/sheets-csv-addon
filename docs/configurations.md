# Configurations

## Default CSV (csv_rfc4180)

Compliant with the [RFC 4180](https://datatracker.ietf.org/doc/html/rfc4180) standard.

- Field delimiter: `,`
- Record delimiter: `\n`
- Enclosing: `"`

The following rules are apllied to all field values:
- Replace `"` with `""`
